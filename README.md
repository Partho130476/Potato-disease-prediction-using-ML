# Potato-disease-prediction-using-ML


This project detects potato leaf diseases using a trained deep learning model.

Technologies:
- Python
- TensorFlow / Keras
- Flask
- HTML

Features:
- Upload potato leaf image
- Model predicts disease type
- Web interface using Flask

Dataset: Potato leaf disease dataset
Model: CNN trained and saved as model1.h5

Datasets:https://www.kaggle.com/datasets/aarishasifkhan/plantvillage-potato-disease-dataset
